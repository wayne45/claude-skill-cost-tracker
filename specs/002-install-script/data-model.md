# Data Model: Cost Tracker Install Script

**Feature**: 002-install-script | **Date**: 2026-03-20 (updated from 2026-03-05)

## File Inventory

The installer is a self-contained script with all file contents embedded as heredocs. No source files are copied from a cloned repo.

### Files Written (from embedded heredocs)

| Target (project-relative) | Permissions | Action on Update |
|--------------------------|-------------|-----------------|
| `.claude/hooks/cost-tracker.sh` | 755 (executable) | Backup `.bak` then overwrite |
| `.claude/commands/cost-report.md` | 644 | Backup `.bak` then overwrite |
| `.claude/commands/cost-session.md` | 644 | Backup `.bak` then overwrite |
| `.claude/commands/cost-reset.md` | 644 | Backup `.bak` then overwrite |
| `.claude/cost-data/pricing.json` | 644 | Backup `.bak` then overwrite |

### Files Merged (not overwritten)

| Target (project-relative) | Action |
|---------------------------|--------|
| `.claude/settings.local.json` | Merge hook entry into existing JSON; create if absent |

### Files Never Touched

| Target (project-relative) | Reason |
|---------------------------|--------|
| `.claude/cost-data/sessions.jsonl` | User's accumulated cost data |
| `.claude/cost-data/report.md` | Generated report (regenerated by /cost-report) |

### Directories Created

| Path | Created When |
|------|-------------|
| `.claude/` | If not present (with confirmation) |
| `.claude/hooks/` | Always (mkdir -p, no-op if exists) |
| `.claude/commands/` | Always (mkdir -p, no-op if exists) |
| `.claude/cost-data/` | Always (mkdir -p, no-op if exists) |

## Settings Schema

### settings.local.json Structure

The installer merges into this structure, preserving all existing content:

```json
{
  "permissions": {
    "allow": ["...existing entries preserved..."]
  },
  "hooks": {
    "Stop": [
      "...existing Stop hooks preserved...",
      {
        "matcher": "",
        "hooks": [
          {
            "type": "command",
            "command": "\"$CLAUDE_PROJECT_DIR\"/.claude/hooks/cost-tracker.sh",
            "timeout": 30000
          }
        ]
      }
    ],
    "...other hook types preserved..."
  },
  "...other top-level keys preserved..."
}
```

### Merge Rules

1. **Top-level keys**: All preserved. Only `hooks` is touched.
2. **hooks object**: All hook types preserved. Only `hooks.Stop` is modified.
3. **hooks.Stop array**: Existing entries preserved. Cost tracker entry added/replaced.
4. **Dedup rule**: If an existing Stop entry contains a hook with command matching `cost-tracker.sh`, it is replaced (not duplicated).

## Installation State Machine

```
┌─────────────┐
│ detect_state │
└──────┬──────┘
       │
       ├── 0 files found ──────→ FRESH_INSTALL
       │                          → proceed directly
       │
       ├── 1-4 files found ────→ PARTIAL_INSTALL
       │                          → warn + backup existing + repair
       │
       └── 5 files found ─────→ ALREADY_INSTALLED
                                  → backup existing + update
```
